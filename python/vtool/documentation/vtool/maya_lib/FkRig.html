VTOOL Rig Classes

FkRig( BufferRig )

The fk rig will create controls on the joints that affect each other in hierarchy.

Each control has the following hiearchy: xform group, driver group, control

Initialize

  • FkRig( description, side )

description is used when naming the controls to give a unique name.

side can be 'C', 'L', 'R'

Example

joints = ['joint_arm_L', 'joint_elbow_L', 'joint_wrist_L']

rig = FkRig( 'arm', 'L' )

rig.set_joints(joints)

rig.set_buffer(True) #Setting buffer True allows us to layer on ik later on.

rig.create()

Functions

set_control_shape( shape_name )

Sets the control to have a shape from the curve library.

create( )

Creates the rig.

set_joints( joints )

Set which joints the rig should work with. Some rigs also accapt transform nodes.

Accepts a list or a string.

set_buffer( bool_value )

bool_value = Set True or False. This specifies wether or not the supplied joints should be constrained with a buffer chain. The buffer chain helps when setting up ik/fk setups. The first joint will have a switch attribute that can be used to switch between rigs.

set_parent( parent )

Specify the parent for the controls group

set_control_size( value )

Sets the size of the controls when created.

set_match_to_rotation( bool_value )

get_drivers()

Returns a list of transforms that are the drivers above controls.
