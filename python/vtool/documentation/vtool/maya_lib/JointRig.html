VETALA Rig Classes

JointRig( Rig )

joints can be a list of joints or the name of a single joint.

Initialize

  • JointRig( description, side )

description is used when naming the controls to give a unique name.

side can be 'C', 'L', 'R'

Functions

set_control_shape( shape_name )

Sets the control to have a shape from the curve library.

create( )

Creates the rig.

set_joints( joints )

Set which joints the rig should work with.

Accepts a list or a string.

Some rigs also accept transform nodes.
