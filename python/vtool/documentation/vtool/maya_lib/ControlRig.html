VETALA Rig Classes

ControlRig( Rig )

This class is meant to build blendshape slider curves that make up a face icon.

Initialize

  • ControlRig( description, side )

description is used when naming the controls to give a unique name.

side can be 'C', 'L', 'R'

Example

from vtool.maya_lib import util


rig = ControlRig( 'blends', 'C' )

rig.set_transforms( ['locator_blends'] )

rig.set_control_count_per_transform( 3 )


rig.set_control_shape( 0, 'circle' )

rig.set_control_shape( 1, 'coffee_mouth' )

rig.set_control_shape( 2, 'coffee_eyes' )


rig.set_control_description(1, 'mouth')

rig.set_control_description(2, 'eyes')

rig.create()

Functions

create( )

Creates the controls.

set_transforms( transforms )

Specify a list of transforms to build controls at their position.

set_control_count_per_transform( int_value )

This allows the user to set multiple control per transform. Beneficial for facial icons that need the controls to pivot from same place.

set_control_shape( index, shape_name )

Set the shape of the control at the specified index. Index corresponds to the number of controls per transform.
