Rig( object )

Base class for rigs.

Functions

set_control_shape( shape_name )

Sets the control to have a shape from the curve library.

create( )

To be implemented by a sub class to create the rig.

JointRig( Rig )

joints can be a list of joints or the name of a single joint.

Functions

set_joints( joints )

Set which joints the rig should work with. Some rigs also accapt transform nodes.

Accepts a list or a string.

BufferRig( JointRig )

Create a buffer chain based on the specified joints. This chain can be used to swith between different rigs. For example an ik rig and an fk rig.

Functions

set_buffer( bool_value )

bool_value = Set True or False

SparseRig( JointRig )

The sparse rig will create controls on joints. The controls are not interconnected. When the rig respects side it will name and color the controls according to their side.

Functions

set_scalable( bool_value )

Specify if the scale from control to joint should also be connected.

set_respect_side( bool_value, tolerance = 0.001)

bool_value = Set True or False

tolerance = The tolerance of what is considered centered.

SparseLocalRig( SparseRig )

The sparse local rig will create controls on joints. The controls are not interconnected. Because it is local the controls can move has a group without affecting the joints. Allowing the joints to say local to the origin. The sparse local rig uses a transform node to buffer it's affect on the joint.

Functions

set_local_constraint( bool_value )

If set true, controls will use a constraint between the control and a buffer group that gets connected into the joint. Otherwise the control will be connected directly into the buffer group. By default this is True.

set_control_to_pivot( bool_value )

When setting the rig to work on cluster transforms using set_joints, this will make sure that the controls will match the pivot of the transform, not just its translate. This function will most likely be removed in the future, and dealt with automatically.

set_local_parent( local_parent ):

This will cause the local buffer to move with the specified transform.

ControlRig( Rig )

This class is meant to build blendshape slider curves that make up a face icon.

Functions

set_transforms( transforms )

Specify a list of transforms to build controls at their position.

set_control_count_per_transform( int_value )

This allows the user to set multiple control per transform. Beneficial for facial icons that need the controls to pivot from same place.

set_control_shape( index, shape_name )

Set the shape of the control at the specified index. Index corresponds to the number of controls per transform.

GroundRig( JointRig )

The ground rig creates 3 controls around the specified joint or transform. Meant to be used at the base controls for a rig.

FkRig( BufferRig )

The fk rig will create controls on the joints that affect each other in hierarchy.

Functions

set_parent( parent )

Specify the parent for the controls group

set_control_size( value )

Sets the size of the controls when created.

set_match_to_rotation( bool_value )

get_drivers()

Returns a list of transforms that are the drivers above controls.

FkLocalRig( FkRig )

The fk local rig will create controls on the joints that affect each other in hierarchy. Because it is local the controls can move has a group without affecting the joints. Allowing the joints to say local to the origin. The fk local rig uses a transform node to buffer it's affect on the joint.

Functions

set_local_parent( local_parent ):

This will cause the local buffer to move with the specified transform.

FkScaleRig( FkRig )

This rig will create fk controls with scalability.

FkCurlNoScaleRig( FkRig )

This will create an fk rig with curl attributes.

Functions

set_curl_axis( axis_letter )

Sets the axis to curl on. Axis letter can equal: 'x', 'y', 'z', 'All'

set_attribute_control( control_name )

Name of the control to add curl attribute(s) to.

set_attribute_name( attribute_name )

Specify a description for the curl attribute.

set_skip_increments( increments )

Specify a list of inices that should be skipped. Indices correspond to control increment. For example, if you want to skip the first control in the hieararchy you would provide: [0].

SimpleFkCurveRig( FkCurlNoScaleRig )

This class will create a fk rig the move joints with a curve. Controls by default are not oriented to anything.

Functions

set_control_xform( vector, inc )

vector is a the number list, eg: [1,0,0], that specifies the movement of the control at the increment inc.

set_orient_joint( joint )

Specifiy the transform to orient controls to.

set_orient_controls_to_joints( bool_value )

Specify wether or not the controls should orient to the nearest joint. Default is False

set_advanced_twist( bool_value )

Specify wether or not to use first and last control twisting. Default is True.

set_control_count( int_value, span_count = None, wire_hires = False )

Specify the number of controls along the curve. Default is 3. Also how man cvs are on the curve and wether the different cv count curve should be wire deformed. Usefull for doing hair dynamics.

set_sub_control( bool_value )

Add sub controls. Default is True.

set_stretchy( bool_value )

Add stretchiness to the joints. Default is True. Does not work when set_ribbon is true.

set_stretch_on_ff( bool_value )

Specify if the stretchy should have a slider to turn off and on. Default is True

set_curve( curve )

Specify the curve to rig to instead of auto generating it.

set_ribbon( bool_value )

Use a ribbon instead of a curve.

set_ribbon_offset( float_value )

Specify the thickness of the ribbon.

set_ribbon_offset_axis( axis_letter )

Specify the axis to offset the ribbon on. Options include: 'X', 'Y', 'Z'

IkAppendageRig( BufferRig )

This class will create a ik rig the move joints like an arm or leg. It has a top and btm control that act on the first and last joint. The other joints pull along with them.

Functions

set_create_twist( bool_value )

Specify if pole vector control should have a twist attribute. Default is True.

set_create_stretchy(bool_value )

Specify if the joints in the ik chain should be strechable. Default is True.

set_pole_offset( value )

Set the value to offset the pole vector control from the middle joint.

set_pole_angle_joints( joints )

Specify which joints in the chain should be used to calculate the pole vector. Provide a list of joint names. This is meant to be used for joint chains that have more than 3 joints.

set_right_side_fix( bool_value )

Specify wether a rig with side 'R' should have special considerations to deal with orientation.

set_orient_constrain( bool_value )

Specify if the last joint in the ik chain should have an orient constraint. Default is True.

set_create_sub_control( bool_value )

Specify if a sub control for the btm control should be created.

set_create_world_switch( bool_value )

Specify wether an attribute to switch the wrists to align with the world plane should be set. For characters in T-Pose this should be set False. For Characters in A-Pose it should be set True. Default is True.

set_top_control_as_locator( bool_value )

Instead of creating a control on the top joint, create a locator. Default is False.

set_match_btm_to_joint( bool_value )

Specify wether the btm control should be matched to the btm joint. Default is True.

set_create_top_control( bool_value )

Create top control. Default is True.
