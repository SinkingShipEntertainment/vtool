VETALA Rig Classes

GroundRig( JointRig )

The ground rig creates 3 controls around the specified joint or transform. Meant to be used as the base controls for a rig.

Initialize

  • GroundRig( description, side )

description is used when naming the controls to give a unique name.

side can be 'C', 'L', 'R'

Example

from vtool.maya_lib import util


rig = GroundRig( 'ground', 'C' )

rig.set_joints('locator_ground')

rig.create()

Functions

set_control_shape( shape_name )

Sets the control to have a shape from the curve library.

create( )

Creates the rig.

set_joints( joints )

Set which joints the rig should work with.

Accepts a list or a string.

Some rigs also accept transform nodes.
